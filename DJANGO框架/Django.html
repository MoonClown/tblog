# Django框架搭建一个应用

# 0. 基本使用

  • 下载
    • 命令行
 pip install django==1.11.25 -i https://pypi.tuna.tsinghua.edu.cn/simple/

  • 创建项目

    • 命令行
     django-admin startproject 项目名称

    • pycharm创建

      file _> new project _> django _> 输入项目的目录 _> 选择解释器 _> create

  • 启动项目

    • 命令行

      切换到项目的根目录下

      python manage.py runserver # 127.0.0.1:8000

      python manage.py runserver 80 # 127.0.0.1:80

      python manage.py runserver 0.0.0.0:80 # 0.0.0.0:80

    • pycharm

      选择django项目 点绿三角 (切记 没有右键启动项目)

# 使用 ORM（数据库）

  1. 创建数据库
creat database 数据库名称

  1. 在 settings 中配置

    DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',  # 连接什么样的数据库
        'NAME': 'bookmanager',			# 数据库的名称
        'HOST': '127.0.0.1',			# IP
        'PORT': 3306,				    # 端口	
        'USER': 'root',					# 用户名
        'PASSWORD': "tong1234"			# 密码
    }
}

#替换MySQLdb模块

  2. 在 __init 文件中添加

    import pymysql
pymysql.install_as_MySQLdb()

  3. 创建表、数据

from django.db import models

# Create your models here.
class User(models.Model):
    username = models.CharField(max_length=32)
    password = models.CharField(max_length=32)


class Publisher(models.Model):
    name = models.CharField(max_length=32)
  1. 生成迁移文件、在命令行下：
python manage.py  makemigrations  #检测已经注册app下的models.py的变更记录
python manage.py migrate   #数据库迁移 

python3 manage.py  makemigrations  #检测已经注册app下的models.py的变更记录
python3 manage.py migrate   #数据库迁移
  1. 表结构已经建成，在 pycharm 中右上角进入 database
  2. 添加 mysql源，填写数据库信息后，下载驱动（最下边的 download）
  3. 在database中添加数据，然后点击提交后可写入到数据库中
